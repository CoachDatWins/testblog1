## Development

1. Clone repo
    ```
    git clone --recurse-submodules git@github.com:zwbetz-gh/zwbetz.git
    ```
1. Change to it
    ```
    cd zwbetz
    ```
1. Install Hugo. Assumes OS is Linux
    ```
    ./task_install_hugo.sh <HUGO_VERSION>
    ```
1. Serve site
    ```
    ./task_site_serve.sh
    ```
1. Build site
    ```
    ./task_site_build.sh
    ```
